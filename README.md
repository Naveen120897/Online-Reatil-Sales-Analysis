# Online Retail Sales Analysis Project(Python)

 ## Overview 

This project aims to analyze online retail sales data using Python. It includes scripts to process sales data, perform analysis, and visualize trends to gain insights into sales performance over time and customer behavior.

## Features

Import and preprocess online sales data

Analyze sales trends over time

Identify top-selling products and Top revenue generating Countries

Visualize sales data using interactive plots and charts

## Dataset

This project uses a dataset containing online sales data capturing the items purchased by customers over a certain period. The dataset is sourced from [here](https://www.kaggle.com/datasets/thedevastator/online-retail-transaction-records). 

## Requirements

Python 3.x

Pandas

Matplotlib

Jupyter Notebook (optional, for interactive analysis)
